3. データフロー構成図
┌─────────────────────────────────────────────────────────────────────────────────┐
│                            データフロー構成                                     │
└─────────────────────────────────────────────────────────────────────────────────┘

【ファイル転送フロー】
オンプレミス → Direct Connect → VGW → NLB → Transfer for SFTP → S3
     │              │           │      │           │              │
     │              │           │      │           │              │
  空調データ      専用線接続   VPC内    負荷分散   SFTP受信      ファイル保存
  (CSV/JSON)     (1Gbps)    ルーティング (高可用性) (認証済み)   (暗号化)

【リアルタイム処理フロー】
S3 → Lambda → DynamoDB → API Gateway → CloudFront → React App
 │      │        │           │            │           │
 │      │        │           │            │           │
ファイル  データ変換  構造化データ  REST API    CDN配信    ユーザー
イベント  (Python)   保存        (認証付き)   (キャッシュ)  インターフェース

【監視・ログフロー】
全サービス → CloudWatch → SNS → 管理者通知
     │           │        │        │
     │           │        │        │
  メトリクス    ログ集約   アラート   Email/SMS
  ログ出力     ダッシュボード 閾値監視   通知
